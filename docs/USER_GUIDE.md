## 一、适用场景

### 这个工具解决什么问题？

本系统帮您在「**组织 → 项目 → 任务**」三层结构下管理团队工作，用于：

| 痛点 | 系统如何解决 |
|------|----------------|
| 事项分散在聊天里，不知谁在做、何时截止 | 每项工作是一条**任务**：有负责人、截止日期、状态（待办/进行中/已完成等） |
| 多人协作交接不清 | 每条任务可有**协助人**；项目内可查**动态**记录 |
| 想看整体进度与时间线 | 提供**甘特图**；项目级**报表**看完成率 |
| 交付文件找不到 | 任务上可上传**交付物**；另有项目**资源中心**汇总筛选 |
| 与同事沟通 | **消息中心**支持与同组织成员的私信（含附件）；任务内另有**讨论区** |

### 谁适合使用？

- **项目负责人 / PM**：创建项目、拆分任务、分配负责人、跟进进度。
- **执行同学**：在「**我的任务**」里看到自己相关的工作；在项目里更新状态、写说明、上传文件。
- **协作成员**：加入项目后被指派任务或设为协助人，使用任务详情与私信沟通。

## 二、快速开始

### 第一步：访问与注册

1. 用浏览器打开管理员给您的产品地址（一般为 `https://…`，需与单位配置一致）。
2. 打开页面上的「**登录**」，切换到「**注册**」（若界面提供）。
3. 按页面提示填写：**邮箱**、**密码**、**姓名**，以及（若要求）**组织名称**。  
4. 提交成功后，一般会**自动登录**并进入您的**组织工作台**。

> **若提示「还没有组织」**：请联系管理员是否在后台为您创建组织或导入演示数据；或由已有组织的同事邀请您加入（视单位流程而定）。

### 第二步：创建项目或加入项目

您需要进入一个**具体项目**，才能创建任务。

**方式 A — 自己新建项目**

1. 在左侧导航栏点击「**项目管理**」。  
2. 在页面中找到「**新建项目**」表单，输入项目名称（例如：`市场部 Q3 专题`）。  
3. 点击创建。成功后通常会**自动进入该项目的工作台**。  

**方式 B — 加入已有项目**

1. 左侧点击「**项目管理**」。  
2. 请项目负责人把**项目 ID**（一长串英文字母与数字）发给您。  
3. 在本页「**加入项目**」输入框中**粘贴项目 ID**，点击加入。  
4. 成功后点击列表中的项目名，进入**项目工作台**。

> 项目负责人可在项目页顶部找到「**复制 ID**」按钮，把 ID 发给同事。

### 第三步：创建任务

进入某个项目的工作台后：

1. 点击右上角「**新建任务**」。  
2. 填写任务标题（必填），可选填写说明、负责人、截止日期等。  
3. 保存后，任务会出现在当前项目的**看板 / 列表 / 甘特**等视图中。

### 第四步：日常最基本流程（建议顺序）

| 顺序 | 您要做的 | 在哪里操作 |
|------|-----------|------------|
| 1 | 查看「今天该做什么」 | 左侧「**我的任务**」：按状态筛选，或点「**去处理**」进入对应项目并打开任务详情 |
| 2 | 更新任务进展 | 进入项目 → 点选任务 → 右侧「任务详情」：改**状态**、写**任务内容**、拖**进度** |
| 3 | 上传交付文件 | 同一侧栏中的「**任务交付物**」：选择文件或拖拽上传 |
| 4 | 与同事私信 | 任务详情里负责人/协助人下方的「**发消息**」，跳转到「**消息中心**」 |

## 三、功能说明

### 3.1 项目管理

- **入口**：左侧「**项目管理**」。  
- **作用**：查看本组织下所有项目、新建项目、通过项目 ID 加入项目。  
- **进入项目**：在项目列表中点击项目名称，进入**项目工作台**（默认多为甘特视图，可切换）。

### 3.2 项目工作台与视图

进入项目后，顶部可切换：

| 视图名称 | 适合做什么 |
|----------|------------|
| **看板** | 按「待办 / 进行中 / 已完成 / 阻塞」分列；可拖拽卡片**改变任务状态**。 |
| **列表** | 表格查看全部任务；可打开任务、删除任务（需权限）。 |
| **甘特** | 按时间轴查看任务起止与进度条，适合排期与对冲风险。 |
| **报表** | 查看本项目任务数量、完成率等汇总。 |
| **动态** | 查看与本项目任务相关的操作记录。 |
| **AI**（智能任务解析） | 粘贴会议纪要或需求文字，系统尝试拆成多条任务草稿；在表格中预览并确认后写入本项目（与看板等视图同源）。单位未开通时按钮可能不可用。 |

另外，顶栏有「**新建任务**」「**资源中心**」：

- **资源中心**：汇总本项目内各任务上传的交付物，支持按任务、类型等筛选（具体筛选项以页面为准）。

### 3.3 任务系统

打开任意一条任务（在任一种视图中点击任务），右侧会出现「**任务详情**」：

- **任务名称 / 任务内容**：可描述目标、验收标准、完成情况说明；支持失焦自动保存或点「保存」。  
- **负责人 / 协助人**：从**已加入本项目**的成员中选择；协助人需先勾选对应成员。  
- **当前状态**：待办、进行中、已完成、阻塞等。  
- **开始日期 / 截止日期**：用于列表排序与甘特展示。  
- **优先级**：便于区分紧急程度。  
- **甘特进度**：0%～100%，影响甘特图进度显示。

**任务讨论**：任务详情中的讨论区用于项目成员围绕本任务留言（与私信不同，私信在「消息中心」）。

### 3.4 文件管理（交付物与私信附件）

**任务交付物（上传文件）**

1. 打开任务详情，找到「**任务交付物**」。  
2. 选择类型（可选），点击「**选择文件**」或将文件**拖入虚线区域**。  
3. 上传完成后可预览（视文件类型）、下载；删除权限取决于您的角色。

> 若页面提示未配置存储：说明管理员尚未配置云端文件服务，此时可能只能看到记录而**无法上传/下载**，请联系管理员。

**资源中心**

1. 在项目工作台顶部点击「**资源中心**」。  
2. 浏览本项目所有任务关联的文件，按需筛选、预览或下载。

**私信中带附件**

1. 左侧打开「**消息中心**」。  
2. 进入与某位同事的会话后，可用「**回形针**」添加附件（对方须与您同属一个组织）。

### 3.5 协作功能

| 功能 | 说明 |
|------|------|
| **我的任务** | 列出与**您相关**的任务（您是负责人，或您是**协助人**）。列表右侧「**去处理**」直接进入项目并打开该任务。 |
| **消息中心** | **私信**：一对一聊天与附件；**通知与提醒**：系统通知列表。 |
| **任务内发消息** | 在负责人、协助人区域使用「**向负责人发消息**」「**向协助人发消息**」，会跳转到消息中心并打开对应会话（不能与自己私聊）。 |
| **邀请同事进项目** | 同事须先加入**同一组织**，再在「项目管理」里用**项目 ID** 加入本项目。 |

### 3.6 智能助手（若单位已开通）

以下功能由管理员为单位统一开通；页面上若显示「未开通」或按钮不可用，表示当前环境尚未启用，您仍可通过「新建任务」、列表与甘特等常规方式工作。

| 入口 | 能做什么 |
|------|-----------|
| **项目工作台 → AI** | 粘贴会议纪要、需求片段或待办清单；系统尝试识别任务名称、内容、负责人与协助人、状态、日期、优先级与进度，生成可预览的表格，确认后一次性写入本项目。 |
| **我的任务** 中的 **智能计划表** | 根据与您相关的任务，生成「今日安排」与「本周节奏」类文字建议，便于排优先级和时间块。 |
| **工作报告** 中的 **智能工作报告** | 根据您当前任务数据，生成今日 / 本周 / 本月工作报告草稿，可在生成结果上修改后再对外使用。 |

## 四、完整示例：筹备「十周年客户答谢活动」

下面是一条从头到尾可照着点的示例，帮助新用户理解各模块如何串起来。

**背景**：您是市场活动负责人，要在 6 周内办完一场线下答谢会，团队共 3 人（您本人 + 设计 + 摄像）。目标是用一个项目把所有事项管起来。

### 1. 建立项目

1. 登录后，左侧点「**项目管理**」。  
2. 「新建项目」名称填写：`十周年客户答谢活动`。  
3. 创建成功后进入该项目工作台。

### 2. 创建关键任务（示例）

点击「**新建任务**」，依次建立（标题示例）：

| 任务标题 | 负责人建议 | 截止日期建议 |
|----------|------------|----------------|
| 确定场地与预算 | 您本人 | 第 1 周末 |
| 主视觉与邀请函设计 | 设计同事 | 第 2 周末 |
| 嘉宾名单与邀约 | 您本人 | 第 3 周末 |
| 活动流程与主持人稿 | 您本人 | 第 4 周末 |
| 现场签到与物料 | 您本人 | 活动前 3 天 |
| 摄影摄像与成片交付 | 摄像同事 | 活动后 5 天内 |

每条任务可在详情里写清**验收标准**（写在「任务内容」里）。

### 3. 使用看板推进

1. 切换到「**看板**」视图。  
2. 开场前多数任务放在「**待办**」或「**进行中**」。  
3. 随着完成，把卡片拖到「**已完成**」（或直接打开任务改状态）。

### 4. 用甘特把控时间

1. 切换到「**甘特**」。  
2. 检查各任务的**开始/截止**是否合理，有无重叠冲突。  
3. 及时调整截止日期或拆分任务。

### 5. 文件与沟通

1. 「主视觉」任务下由设计上传 PDF/图片到「**任务交付物**」。  
2. 您在任务详情通过「**向负责人发消息**」（若对方不是您）或使用「**消息中心**」与设计确认修改意见。  
3. 活动结束后，摄像在对应任务上传成片链接或大文件（若存储已配置）。

### 6. 例会前自查

1. 打开「**报表**」看整体完成率。  
2. 打开「**资源中心**」确认所有成品文件都能查到。

## 五、常见问题 FAQ

以下为真实界面操作步骤；若与您单位定制不符，请以管理员说明为准。

### Q1：登录后页面空白或立刻跳出登录？

**可能原因**：浏览器地址与系统配置的站点地址不一致（例如用了 `http` 而要求是 `https`，或域名/端口不同）。

**建议步骤**：

1. 复制管理员提供的**完整网址**（含 `https://`），新开一个浏览器标签页粘贴访问。  
2. 清除本站点的 Cookie 后重试登录。  
3. 仍不行：换 Chrome / Edge 最新版本，关闭拦截第三方 Cookie 的插件后再试。

---

### Q2：提示「项目不存在或不属于本组织」，无法加入项目？

**原因**：项目 ID 填错，或该项目不属于您当前所在的组织。

**建议步骤**：

1. 请项目负责人重新发一遍**项目 ID**（一串字母数字，可从项目页「复制 ID」获得）。  
2. 确认您与对方在**同一个组织**（左侧顶部组织名称）。  
3. 在「项目管理 → 加入项目」中**重新粘贴** ID，勿多空格。

---

### Q3：「我的任务」里没有某条任务，但负责人明明是我？

**建议排查**：

1. 打开该项目 → 找到该任务 → 看「**负责人**」下拉框是否真的是您（有时被改成了别人）。  
2. 若您是**协助人**而不是负责人：只要任务里勾选了您为协助人，也会出现在「我的任务」里；若两者都不是，则不会出现在「我的任务」。  
3. 点击左侧「我的任务」顶部不同**状态**标签（待办/进行中/…），避免被筛选隐藏。

---

### Q4：任务交付物上传失败或提示未配置存储？

**说明**：上传文件依赖管理员在服务器侧配置云端存储。

**您可以做的**：

1. 截图完整报错文案发给管理员。  
2. 暂时把文件通过「消息中心」私信发给同事（若私信附件可用），并在「任务内容」里写明文件说明与链接。

---

### Q5：私信发不出去 / 找不到对方？

**常见规则**：私信双方必须都是**同一组织**的成员。

**建议步骤**：

1. 确认对方已注册并加入了**您的组织**（不是只加了项目而未加入组织）。  
2. 在「消息中心 → 私信」从左侧会话列表选择；若尚无会话，可从任务详情点「**向负责人发消息**」会自动尝试打开会话。  
3. 若对方不在组织：请组织管理员先把对方加入组织，再通过「项目管理 → 加入项目」拉进具体项目。

---

### Q6：忘记密码怎么办？

**建议步骤**：

1. 在登录页找到「**忘记密码**」链接（若您的单位开放了邮件重置）。  
2. 输入注册邮箱，查收邮件并按链接设置新密码。  

> 若页面提示无法发送邮件：说明单位未配置邮件服务，请联系管理员**人工重置**或使用单位规定的账号找回渠道。

---

### Q7：甘特图上某条任务时间不对？

**建议步骤**：

1. 打开该任务详情，检查「**开始日期**」「**截止日期**」是否正确。  
2. 检查任务「**状态**」是否为「已完成」却仍横跨很长时间——若已结束，可将截止日期调到实际结束日，便于报表准确。

---

### Q8：拖拽看板任务没反应？

**建议步骤**：

1. 确认您对该任务有**编辑权限**（一般为项目成员中的非访客角色；具体以单位规则为准）。  
2. 刷新页面后重试；仍不行请改用打开任务详情，手动修改「**当前状态**」。

---

### Q9：智能助手相关按钮是灰色的或提示未开通？

**说明**：智能助手由单位管理员在后台开通；未开通时不影响您使用看板、列表、甘特与手动新建任务。

**建议**：向管理员确认本单位是否已启用；未启用期间请用「新建任务」等方式照常录入与跟进。

---

### Q10：想导出 Excel / 报表打印？

**说明**：当前产品以在线协作为主；若界面未提供「导出」按钮，可把「列表」视图中的任务信息复制到 Excel，或使用浏览器打印（部分浏览器支持打印当前页）。
