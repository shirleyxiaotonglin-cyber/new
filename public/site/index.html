核心能力

以下与 Next.js 应用内已实现能力对应；静态页仅作展示，以登录后界面为准。

  • 📋

    项目与任务管理

    按组织、项目管理工作；看板与列表视图，任务侧栏维护状态、优先级、负责人与协助人、子任务、依赖与标签，并支持「我的任务」汇总。

  • 📊

    甘特图系统

    时间轴展示起止、进度与依赖，与看板、列表数据同源，便于排期与对照。

  • 👥

    多人协作

    组织与项目成员及角色；项目在线协作者提示，任务分配与活动同步。

  • 💬

    评论与消息

    任务内讨论与项目动态；组织内私信与消息中心，便于跟进与沟通。

  • 📁

    文件与资源中心

    任务侧上传交付物；项目「资源中心」汇总全项目文件，支持预览与下载（需配置对象存储）。

  • ✨

    AI 智能助手

    在项目内将自然语言解析为多条任务草稿并写入（需配置 OpenRouter）；与手动建任务并存。

运行完整应用（Node.js）

本页为静态 HTML；下列命令在项目根目录执行，可与本 static-site 一起部署在同一域名下。 

npm install
npx prisma migrate dev
npm run db:seed
npm run dev
# 浏览器 http://localhost:3000
